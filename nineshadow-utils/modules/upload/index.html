文件上传

文件上传模块提供了单文件上传、批量上传、图片压缩等功能，支持上传进度监控。

API 列表

Section titled “API 列表”
方法说明返回类型
getUploadSignature(data)获取上传签名Promise<UploadSignatureResponse>
uploadFile(file, options?)上传单个文件Promise<string>
uploadFiles(files, options?)批量上传文件Promise<string[]>
uploadImage(file, maxWidth?, maxHeight?, quality?, options?)上传图片（带压缩）Promise<string>

压缩参数说明

Section titled “压缩参数说明”
  • maxWidth: 图片最大宽度，超过会等比缩放
  • maxHeight: 图片最大高度，超过会等比缩放
  • quality: 压缩质量，范围 0-1，推荐 0.8
    • 1.0 = 原始质量（无压缩）
    • 0.8 = 高质量（推荐）
    • 0.5 = 中等质量
    • 0.3 = 低质量

Section titled “最佳实践”
  1. 文件大小限制：在前端检查文件大小，避免上传过大文件
  2. 文件类型检查：限制允许上传的文件类型
  3. 进度显示：使用进度回调显示上传进度
  4. 图片压缩：上传图片时使用 uploadImage 进行压缩
  5. 错误处理：提供友好的错误提示
