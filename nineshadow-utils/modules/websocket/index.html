自动重连机制

Section titled “自动重连机制”

WebSocket 客户端内置了自动重连机制：

  1. 连接断开时自动重连
  2. 重连间隔由 reconnectInterval 控制
  3. 最大重连次数由 maxReconnectAttempts 控制
  4. 超过最大重连次数后停止重连

心跳检测机制

Section titled “心跳检测机制”

WebSocket 客户端内置了心跳检测机制：

  1. 定期发送心跳包保持连接
  2. 心跳间隔由 heartbeatInterval 控制
  3. 如果服务器无响应会触发重连

最佳实践

Section titled “最佳实践”
  1. 组件卸载时断开连接：避免内存泄漏
  2. 监听连接状态：根据连接状态显示 UI
  3. 错误处理：提供友好的错误提示
  4. 消息类型：使用明确的消息类型区分不同业务
  5. 心跳配置：根据实际需求调整心跳间隔
